Features
  • Autocomplete tool
  • No excess requests to server
  • Wrap on standart input
  • Results list view definition
  • Definition of field for input
Getting started
Insert in head
<!-- basic style -->
<link rel="stylesheet" type="text/css" href="your_path/css/mprompter.css" />
<!-- required javascripts -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.1/jquery.min.js"></script>
<script type="text/javascript" src="your_path/js/mprompter.js"></script>
Insert in body
<input type="text" id="my_search" style="width:300px;" />

<script type="text/javascript">
$(document).ready( function(){
  $('#my_search').mPrompter({
    query_url	: 'path_to_server_script',
    item	: '#{id} <b>{name}</b><br>{description}',
    value	: '{name}',
  });
});
</script>
Documentation
Options
symbolsType: integer, Default: 3 Count of entered symbols for request to server heightType: integer, Default: 200 Maximum height for popup list of results. itemType: string, Default: "{name}" Template to imaging item in list of results. Fields defined as {fieldname} valueType: string, Default: "{name}" Template to imaging result in input field. Fields defined as {fieldname} query_urlType: string, Default: "./search.php" URL of server-side script, which returns resut in JSON format query_paramType: string, Default: "q" Name of variable in search request. paramsType: object, Default: {} Additional your parameters for request to server.
Methods
no methods
Events
onQuery( params:object ) Triggered in mPrompter before request to server, contains object of sended parameters onSuccess( message:string, data:array of objects ) Triggered on succesfull query, contains message and array of founded by query items from server responce onError( message:string ) Triggered on failed query, contains message about error onSelect( item:object ) Triggered when is selectes item from popup list, contains this item
